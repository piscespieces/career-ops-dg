# How to Customize the System to Your Career

> **Coming soon.** This guide is planned. In the meantime, open Claude Code and say: *"Update my profile with this information: [paste your details]."*

This guide explains how to teach Career-Ops about you so that evaluations feel accurate from the start.

**What this covers:**

- Editing archetypes to match your target career track
- Adjusting scoring weights to reflect your priorities (remote policy, comp, tech stack)
- Writing a strong narrative and exit story in `config/profile.yml`
- Setting deal-breakers so low-fit roles are filtered out automatically
- Updating `modes/_profile.md` with custom evaluation instructions
