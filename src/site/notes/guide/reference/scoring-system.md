# Scoring System Reference

> **Coming soon.** This reference is planned. In the meantime, Block A through F are described in the evaluation report itself — open any file in `reports/` to see the structure.

A complete reference for the 6-block evaluation system. Use this page to understand what each block measures, how the 0–5 score is calculated, and what a score in a given range means.

**What this covers:**

- How the overall score (0.0–5.0) is computed from weighted dimensions
- **Block A** — Role summary: archetype, domain, function, seniority, remote policy, team size
- **Block B** — CV match: requirement-by-requirement table, gaps, and severity ratings
- **Block C** — Level strategy: how to position yourself and whether to apply at level or one above/below
- **Block D** — Compensation research: market rate comparison, geographic discount analysis
- **Block E** — Personalization: application angle, subject line, specific talking points
- **Block F** — Interview prep: STAR+R stories generated from your CV for this role
- **Block G** — Posting legitimacy: tier classification (verified, likely active, unconfirmed)
- Score thresholds: what 4.0+, 3.0–3.9, and below 3.0 mean in practice
