# Profile Fields Reference

> **Coming soon.** This reference is planned. In the meantime, see `config/profile.example.yml` for all available fields with inline comments.

A complete reference for every field in `config/profile.yml`. Use this page when you want to know exactly what a field does and how Career-Ops uses it during evaluations.

**What this covers:**

- `candidate` — personal details, contact info, and optional Canva design ID
- `target_roles.primary` — your North Star job titles
- `target_roles.archetypes` — role archetypes and their fit levels (primary / secondary / adjacent)
- `narrative` — headline, exit story, superpowers, and proof points
- `compensation` — target range, currency, minimum, and location flexibility
- `location` — country, timezone, visa status, and on-site availability
